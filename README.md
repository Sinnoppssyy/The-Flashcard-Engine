# 🧠 FlashMind — AI Flashcard Engine

An AI-powered web application that converts PDFs into smart flashcards and helps users learn using spaced repetition and active recall.

---

## 🚀 Features

- 📄 Upload PDF and generate flashcards instantly  
- 🤖 AI-generated questions & detailed answers  
- 🧠 Spaced repetition learning system  
- 📝 Quiz mode with MCQs  
- 🎮 Interactive matching game  
- 📊 Track learning progress  
- ⚡ Fast and responsive UI  

---

## 🛠️ Tech Stack

- Frontend: Next.js (React)  
- Backend/API: Next.js API Routes  
- AI: Groq API (Llama 3)  
- Styling: CSS  
- Deployment: Vercel

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

git clone https://github.com/Sinnoppssyy/The-Flashcard-Engine.git  
cd The-Flashcard-Engine  

---

### 2️⃣ Install dependencies

npm install  

---

### 3️⃣ Setup environment variables

Create a `.env.local` file in the root directory:

GROQ_API_KEY=your_api_key_here  

⚠️ Important:  
- Do NOT commit `.env.local`  
- Keep your API key secure  

---

### 4️⃣ Run the application

npm run dev  

App will run on:  
(https://the-flashcard-engine-rust.vercel.app/)

---

## 🌐 Deployment

This project can be easily deployed using Vercel:

1. Push code to GitHub  
2. Import project in Vercel  
3. Add environment variables in Vercel dashboard  
4. Deploy 🚀  

---

## 🔄 How It Works

1. Upload a PDF 📄  
2. Extract text from document  
3. AI generates flashcards 🤖  
4. Study using smart learning modes 🧠  

---

## 🔒 Security Notes

- API keys are stored in `.env.local`  
- `.env.local` is ignored using `.gitignore`  
- Never expose API keys in frontend code
